# Hilik's Pizza Lab

## הפעלה במחשב
חלץ את הקובץ והריץ בתיקייה: `python -m http.server 8080` ואז פתח `http://localhost:8080`.

## התקנה באייפון
פרסם את התיקייה ב-GitHub Pages / Cloudflare Pages / Vercel, פתח ב-Safari ובחר Share > Add to Home Screen.

ברירת המחדל: 63.6% הידרציה, 21.2% קמח בפוליש, 2.12% מלח, 0.085% שמרים יבשים, 72 שעות.
